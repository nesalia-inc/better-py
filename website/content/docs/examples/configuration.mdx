# Configuration Management

Immutable configuration with better-py.

## Code

```python
from better_py import PersistentMap

class Config:
    def __init__(self, data: PersistentMap = None):
        self.data = data or PersistentMap.empty()
    
    def get(self, key: str, default=None):
        return self.data.get(key).unwrap_or(default)
    
    def set(self, key: str, value) -> 'Config':
        return Config(self.data.set(key, value))
    
    def merge(self, other: 'Config') -> 'Config':
        return Config(self.data.update(other.data))

# Usage
config = Config()
config = config.set("host", "localhost")
config = config.set("port", 8080)
```
