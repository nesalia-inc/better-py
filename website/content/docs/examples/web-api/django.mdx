# Django Integration

Use better-py within Django views.

## Example

```python
from django.http import JsonResponse
from better_py import Result

def user_view(request, user_id: int):
    result = fetch_user(user_id)
    
    if result.is_ok():
        return JsonResponse({"user": result.unwrap().to_dict()})
    else:
        return JsonResponse(
            {"error": result.unwrap_error()},
            status=404
        )
```
