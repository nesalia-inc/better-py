Build robust APIs with FastAPI and better-py.

## Example

```python
from fastapi import FastAPI
from better_py import Result

app = FastAPI()

@app.get("/users/{user_id}")
async def get_user(user_id: int) -> dict:
    result = await fetch_user(user_id)
    
    if result.is_ok():
        return {"user": result.unwrap().to_dict()}
    else:
        raise HTTPException(404, detail=result.unwrap_error())
```
