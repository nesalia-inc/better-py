---
title: SQLAlchemy Integration
description: Using better-py with SQLAlchemy
---

Use better-py with SQLAlchemy for database operations.

## Example

```python
from better_py import Result
from sqlalchemy.orm import Session

def get_user(session: Session, user_id: int) -> Result[User, str]:
    user = session.query(User).filter(User.id == user_id).first()
    if not user:
        return Result.error(f"User {user_id} not found")
    return Result.ok(user)
```
