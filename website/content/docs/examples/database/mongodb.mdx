---
title: MongoDB Integration
description: Using better-py with MongoDB
---

Use better-py with MongoDB (motor).

## Example

```python
from better_py import AsyncResult
from motor.motor_asyncio import AsyncIOMotorClient

async def get_user(db: AsyncIOMotorClient, user_id: int) -> AsyncResult[dict, str]:
    user = await db.users.find_one({"_id": user_id})
    if not user:
        return AsyncResult.error("User not found")
    return AsyncResult.ok(user)
```
