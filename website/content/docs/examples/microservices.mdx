Service-to-service communication with better-py.

## Example

```python
from better_py import Result, AsyncResult

async def fetch_user_data(user_id: int) -> AsyncResult[dict, str]:
    user = await user_service.get(user_id)
    profile = await profile_service.get(user_id)
    
    if user.is_error():
        return AsyncResult.error(user.unwrap_error())
    
    return AsyncResult.ok({
        "user": user.unwrap(),
        "profile": profile.unwrap()
    })
```
