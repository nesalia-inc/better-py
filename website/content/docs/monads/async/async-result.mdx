---
title: AsyncResult Monad
description: Asynchronous error handling
---

Async version of the Result monad for operations that can fail.

---

## Overview

```python
from better_py import AsyncResult

async def fetch_user(id: int) -> AsyncResult[User, str]:
    try:
        user = await api.get(f"/users/{id}")
        return AsyncResult.ok(user)
    except HTTPError as e:
        return AsyncResult.error(str(e))
```

---

## Creating AsyncResult

### AsyncResult.ok()

```python
AsyncResult.ok(value)
```

### AsyncResult.error()

```python
AsyncResult.error("Something failed")
```

---

## Transforming AsyncResult

### map()

```python
(await fetch_user(id)).map(lambda u: u.name)
```

### bind()

```python
(await fetch_user(id)).bind(lambda u: fetch_profile(u.id))
```

---

## Real-World Example

```python
from better_py import AsyncResult

async def fetch_user_profile(id: int) -> AsyncResult[dict, str]:
    return (
        await fetch_user(id)
        .bind(lambda user: fetch_profile(user.id))
        .map(lambda profile: {
            "name": user.name,
            "bio": profile.bio
        })
    )

result = await fetch_user_profile(123)
if result.is_ok():
    profile = result.unwrap()
    print(f"Profile: {profile}")
else:
    print(f"Error: {result.unwrap_error()}")
```
