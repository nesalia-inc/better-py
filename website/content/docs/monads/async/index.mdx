# Async Monads

Async versions of the Maybe and Result monads for working with asynchronous code.

---

## Overview

<Callout type="success" title="When to Use Async Monads">

-  Async database queries
-  Async API calls
-  Any async operation that can fail or return no value

</Callout>

---

## Available Async Monads

- **[AsyncMaybe](/docs/monads/async/async-maybe)** - Async optional values
- **[AsyncResult](/docs/monads/async/async-result)** - Async error handling
