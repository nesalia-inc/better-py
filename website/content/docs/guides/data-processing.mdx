# Data Processing Guide

Build robust data processing pipelines.

## Pipeline Pattern

```python
from better_py import Result, PersistentList

def process_pipeline(source: str) -> Result[list, str]:
    return (
        extract_data(source)
        .bind(clean_data)
        .bind(validate_data)
        .map(transform_data)
    )
```

## Error Handling

Each step can fail explicitly:

```python
def extract_data(source: str) -> Result[list, str]:
    try:
        data = read_csv(source)
        return Result.ok(data)
    except Exception as e:
        return Result.error(f"Extract failed: {e}")
```
