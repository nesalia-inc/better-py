Manage application state functionally.

## Immutable State

```python
from better_py import PersistentMap

class AppState:
    def __init__(self):
        self.state = PersistentMap.empty()
    
    def update(self, key: str, value):
        return AppState(self.state.set(key, value))
```

## State Monad

For complex state transitions:

```python
from better_py import State

def increment_and_double() -> State[int, int]:
    return (
        State.modify(lambda n: n + 1)
        .bind(lambda _: State.get())
        .map(lambda n: n * 2)
    )
```
