# Form Validation Guide

Build robust form validation that accumulates all errors.

## Overview

Use the Validation monad to collect all validation errors at once.

## Example: User Registration

```python
from better_py import Validation

def validate_registration(data: dict) -> Validation[dict, str]:
    name_result = validate_name(data.get("name", ""))
    email_result = validate_email(data.get("email", ""))
    password_result = validate_password(data.get("password", ""))
    
    return (
        name_result
        .ap(email_result)
        .ap(password_result)
        .map(lambda parts: {
            "name": parts[0],
            "email": parts[1],
            "password": parts[2]
        })
    )

result = validate_registration({
    "name": "A",
    "email": "bad",
    "password": "weak"
})

if result.is_valid():
    register(result.unwrap())
else:
    show_errors(result.get_errors())
```
