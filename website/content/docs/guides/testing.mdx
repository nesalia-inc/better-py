# Testing Functional Code

Pure functions are easy to test!

## Unit Tests

```python
def test_maybe_map():
    maybe = Maybe.some(5)
    result = maybe.map(lambda x: x * 2)
    assert result == Maybe.some(10)

def test_result_chain():
    result = divide(10, 2).bind(lambda x: Result.ok(x * 2))
    assert result == Result.ok(20)
```

## Property-Based Tests

```python
from hypothesis import given, strategies as st

@given(st.integers())
def test_double_is_always_even(n):
    result = double(n)
    assert result % 2 == 0
```
