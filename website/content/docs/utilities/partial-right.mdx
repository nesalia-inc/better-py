# partial_right

Fix arguments from the right side of a function.

---

## Overview

```python
from better_py import partial_right

def greet(greeting: str, name: str) -> str:
    return f"{greeting}, {name}!"

say_hello = partial_right(greet, "Hello")
result = say_hello("Alice")  # "Hello, Alice!"
```

---

## Real-World Examples

### Default Arguments

```python
from better_py import partial_right

def write_file(path: str, content: str, encoding: str = "utf-8"):
    with open(path, "w", encoding=encoding) as f:
        f.write(content)

write_utf8 = partial_right(write_file, encoding="utf-8")
write_ascii = partial_right(write_file, encoding="ascii")
```
