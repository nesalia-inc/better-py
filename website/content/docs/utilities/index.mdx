## Quick Examples

### compose()

```python
from better_py import compose

# Right-to-left composition
pipeline = compose(str, lambda x: x * 2, lambda x: x + 1)
result = pipeline(5)  # "12" (5 + 1 = 6, 6 * 2 = 12, str(12) = "12")
```

### pipe()

```python
from better_py import pipe

# Left-to-right pipeline
result = pipe(
    5,
    lambda x: x + 1,  # 6
    lambda x: x * 2,  # 12
    str             # "12"
)
```

### curry()

```python
from better_py import curry

@curry
def add(a: int, b: int, c: int) -> int:
    return a + b + c

add_5 = add(5)          # Partial application
add_5_and_3 = add_5(3)  # More partial
result = add_5_and_3(2)  # 10
```
