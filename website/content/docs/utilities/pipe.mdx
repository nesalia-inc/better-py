# pipe & flow

Create left-to-right data pipelines.

---

## pipe()

Pipe a value through functions:

```python
from better_py import pipe

result = pipe(
    5,
    lambda x: x + 1,   # 6
    lambda x: x * 2,   # 12
    str               # "12"
)
```

## flow()

Alias for compose_left:

```python
from better_py import flow

pipeline = flow(lambda x: x + 1, lambda x: x * 2)
result = pipeline(5)  # 12
```

---

## Real-World Examples

### Data Processing

```python
from better_py import pipe

users = [{"name": "alice", "age": 30}, ...]

result = pipe(
    users,
    lambda us: filter(lambda u: u["age"] >= 18, us),
    lambda us: map(lambda u: u["name"], us),
    list
)
# ["alice", ...]
```
