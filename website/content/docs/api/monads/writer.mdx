# Writer API Reference

## Constructor

### Writer(log: W, value: A)
Create with log and value.

## Instance Methods

### map(func: Callable[[A], B]) -> Writer[W, B]
Transform value.

### bind(func: Callable[[A], Writer[W, B]]) -> Writer[W, B]
Chain Writers.

### run() -> tuple[W, A]
Get (log, value) tuple.

## Type Parameters

- W: Log type (list, str, int)
- A: Value type
