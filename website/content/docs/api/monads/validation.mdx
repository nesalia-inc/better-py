# Validation API Reference

## Static Methods

### valid(value: T) -> Validation[T, E]
Create a valid Validation.

### invalid(errors: list[E]) -> Validation[T, E]
Create an invalid Validation with errors.

## Instance Methods

### is_valid() -> bool
Check if valid.

### is_invalid() -> bool
Check if invalid.

### get_errors() -> list[E]
Get error list.

### map(func: Callable[[T], U]) -> Validation[U, E]
Transform valid value.

### ap() -> Validation
Applicative apply (accumulate errors).

## Type Parameters

- T: The valid value type
- E: The error type
