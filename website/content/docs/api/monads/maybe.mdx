# Maybe API Reference

## Static Methods

### some(value: T) -> Maybe[T]
Create a Maybe containing a value.

### nothing() -> Maybe[T]
Create an empty Maybe.

### from_value(value: T | None) -> Maybe[T]
Create from optional value (None â†’ Nothing).

### some_none() -> Maybe[T | None]
Create a Maybe explicitly containing None.

## Instance Methods

### is_some() -> bool
Check if value exists.

### is_nothing() -> bool
Check if empty.

### map(func: Callable[[T], U]) -> Maybe[U]
Transform the value.

### bind(func: Callable[[T], Maybe[U]]) -> Maybe[U]
Chain Maybe-returning operations.

### flat_map(func: Callable[[T], Maybe[U]]) -> Maybe[U]
Alias for bind().

### and_then(func: Callable[[T], Maybe[U]]) -> Maybe[U]
Alias for bind().

### unwrap() -> T
Get the value (raises if Nothing).

### unwrap_or(default: T) -> T
Get value or default.

### unwrap_or_else(supplier: Callable[[], T]) -> T
Get value or compute default.

### or_else(default: Maybe[T]) -> Maybe[T]
Provide alternative Maybe.

## Type Parameters

- T: The contained value type
