# State API Reference

## Static Methods

### get() -> State[S, S]
Get current state.

### put(state: S) -> State[S, None]
Set new state.

### modify(func: Callable[[S], S]) -> State[S, None]
Modify state.

## Instance Methods

### map(func: Callable[[A], B]) -> State[S, B]
Transform output.

### bind(func: Callable[[A], State[S, B]]) -> State[S, B]
Chain State operations.

### run(initial_state: S) -> tuple[A, S]
Run with initial state.

## Type Parameters

- S: State type
- A: Output type
