# Either API Reference

## Constructor Methods

### Either.right(value: R) -> Either[L, R]
Create an Either containing a right (success) value.

### Either.left(value: L) -> Either[L, R]
Create an Either containing a left (error) value.

## Instance Methods

### map(func: Callable[[R], U]) -> Either[L, U]
Transform the right value.

### map_left(func: Callable[[L], L2]) -> Either[L2, R]
Transform the left value.

### swap() -> Either[R, L]
Swap left and right.

### unwrap() -> L | R
Get the value (left or right).

### is_right() -> bool
Check if this is Right.

### is_left() -> bool
Check if this is Left.

## Type Parameters

- L: The left (error) type
- R: The right (success) type
