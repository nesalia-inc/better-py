# IO API Reference

## Constructor

### IO(effect: Callable[[], T])
Wrap a side effect.

## Instance Methods

### map(func: Callable[[T], U]) -> IO[U]
Transform result.

### bind(func: Callable[[T], IO[U]]) -> IO[U]
Chain IO operations.

### run() -> T
Execute the side effect.
