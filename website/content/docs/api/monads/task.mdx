# Task API Reference

## Constructor

### Task(coroutine: Callable[[], Coroutine])
Wrap an async coroutine.

## Instance Methods

### map(func: Callable[[T], U]) -> Task[U]
Transform result.

### bind(func: Callable[[T], Task[U]]) -> Task[U]
Chain Tasks.

### run() -> Coroutine
Execute the task.
