# curry()

Transform function for partial application.

## Signature

```python
def curry(func: Callable) -> Callable:
    ...
```

## Example

```python
@curry
def add(a: int, b: int) -> int:
    return a + b

add_5 = add(5)
result = add_5(3)  # 8
```
