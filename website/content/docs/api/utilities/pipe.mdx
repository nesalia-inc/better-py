# pipe()

Left-to-right data pipeline.

## Signature

```python
def pipe(value: Any, *functions: Callable) -> Any:
    ...
```

## Example

```python
result = pipe(
    5,
    lambda x: x + 1,
    lambda x: x * 2
)  # 12
```
