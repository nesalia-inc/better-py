# Combinable Protocol

## Protocol Definition

```python
@runtime_checkable
class Combinable(Protocol[T]):
    def combine(self, other: Combinable[T]) -> Combinable[T]: ...
```

## Required Methods

### combine(other: Combinable[T]) -> Combinable[T]
Combine two values.

## Type Parameters

- T: The combined type
