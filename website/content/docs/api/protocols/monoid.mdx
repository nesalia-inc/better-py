# Monoid Protocol

## Protocol Definition

```python
@runtime_checkable
class Monoid(Protocol[T]):
    def combine(self, other: Monoid[T]) -> Monoid[T]: ...
    @staticmethod
    def identity() -> Monoid[T]: ...
```

## Required Methods

### combine(other: Monoid[T]) -> Monoid[T]
Combine two values.

### identity() -> Monoid[T]
Return identity element.

## Type Parameters

- T: The monoid type
