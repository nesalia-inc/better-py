# Updatable Protocol

## Protocol Definition

```python
@runtime_checkable
class Updatable(Protocol[T]):
    def update(self, **kwargs) -> Updatable[T]: ...
```

## Required Methods

### update(**kwargs) -> Updatable[T]
Immutable update.

## Type Parameters

- T: The updated type
