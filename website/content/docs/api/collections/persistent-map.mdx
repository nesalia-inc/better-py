# PersistentMap API

## Static Methods

### empty() -> PersistentMap
Create empty map.

### from_dict(data: dict) -> PersistentMap
Create from dict.

## Instance Methods

### get(key: K) -> Maybe[V]
Get value.

### set(key: K, value: V) -> PersistentMap
Set key-value pair.

### delete(key: K) -> PersistentMap
Remove key.

### set_in(path: str, value: Any) -> PersistentMap
Deep update.

### to_dict() -> dict
Convert to dict.
