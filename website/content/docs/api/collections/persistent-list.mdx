# PersistentList API Reference

## Static Methods

### empty() -> PersistentList[T]
Create empty list.

### of(*items: T) -> PersistentList[T]
Create list from items.

### from_iterable(items: Iterable[T]) -> PersistentList[T]
Create from iterable.

## Instance Methods

### is_empty() -> bool
Check if empty.

### prepend(item: T) -> PersistentList[T]
Add item at front (O(1)).

### append(item: T) -> PersistentList[T]
Add item at end (O(n)).

### head() -> Maybe[T]
Get first element.

### tail() -> PersistentList[T]
Get rest of list.

### map(func: Callable[[T], U]) -> PersistentList[U]
Transform all elements.

### filter(pred: Callable[[T], bool]) -> PersistentList[T]
Keep matching elements.

### reduce(func, initial) -> T
Reduce to single value.

### to_list() -> list[T]
Convert to Python list.
