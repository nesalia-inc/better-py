# Monoid Protocol

The `Monoid` protocol extends `Combinable` with an identity element.

---

## Overview

```python
from better_py.protocols import Monoid

class Sum(Monoid):
    def combine(self, other):
        return Sum(self.value + other.value)
    
    @staticmethod
    def identity():
        return Sum(0)
```
