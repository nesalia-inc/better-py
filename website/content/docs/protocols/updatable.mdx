# Updatable Protocol

The `Updatable` protocol represents objects that support immutable updates.

---

## Overview

```python
from better_py.protocols import Updatable

class Config(Updatable):
    def update(self, **kwargs):
        return Config(**{**self.__dict__, **kwargs})
```
