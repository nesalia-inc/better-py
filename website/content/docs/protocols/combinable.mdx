title: Combinable Protocol
description: Objects that can be combined
---

The `Combinable` protocol represents objects that can be combined.

---

## Overview

```python
from better_py.protocols import Combinable

class Money(Combinable):
    def combine(self, other):
        return Money(self.amount + other.amount)
```
