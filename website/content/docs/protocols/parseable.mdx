# Parseable Protocol

The `Parseable` protocol represents objects that can be parsed from text.

---

## Overview

```python
from better_py.protocols import Parseable

class Email(Parseable):
    @staticmethod
    def parse(text: str):
        if "@" in text:
            return Result.ok(Email(text))
        return Result.error("Invalid email")
```
