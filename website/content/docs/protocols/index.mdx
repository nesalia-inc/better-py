# Protocols

better-py protocols define functional patterns using structural typing, inspired by `collections.abc`.

---

## What are Protocols?

Protocols are **interfaces** that any class can implement by having the right methods - no inheritance needed!

```python
from better_py.protocols import Mappable

class MyList:
    def map(self, func): ...

# MyList is Mappable!
```

---

## Available Protocols

| Protocol | Methods | Purpose |
|----------|---------|---------|
| **[Mappable](/docs/protocols/mappable)** | `map()` | Transform values |
| **[Reducible](/docs/protocols/reducible)** | `reduce()` | Reduce to single value |
| **[Combinable](/docs/protocols/combinable)** | `combine()` | Combine two values |
| **[Monoid](/docs/protocols/monoid)** | `combine()`, `identity()` | Combine with identity |
| **[Updatable](/docs/protocols/updatable)** | `update()` | Immutable updates |
| **[Traversable](/docs/protocols/traversable)** | `traverse()` | Traverse with effects |
| **[Parseable](/docs/protocols/parseable)** | `parse()` | Parse from text |
| **[Validable](/docs/protocols/validable)** | `validate()` | Validate data |

## Using Protocols

### Implementing a Protocol

```python
from better_py.protocols import Mappable

class Container(Mappable):
    def map(self, func):
        return Container([func(x) for x in self.items])
```

### Checking Protocol Compliance

```python
from better_py.protocols import Mappable

obj = MyContainer()
if isinstance(obj, Mappable):
    # obj has map()
    result = obj.map(lambda x: x * 2)
```
