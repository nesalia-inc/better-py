---
title: Traversable Protocol
description: Objects traversable with effects
---

The `Traversable` protocol represents objects that can be traversed with monadic effects.

---

## Overview

```python
from better_py.protocols import Traversable

class TraversableList(Traversable):
    def traverse(self, func):
        # Apply function returning monads
        ...
```
