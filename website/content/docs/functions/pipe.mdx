## `pipe(value, *functions)`

Pass a value through a sequence of functions.

```python
from better_py.functions import pipe

pipe(5, lambda x: x + 1, lambda x: x * 2)  # 12
```

## `flow(*functions)`

Create a reusable function from a chain.

```python
from better_py.functions import flow

process = flow(lambda x: x + 1, lambda x: x * 2)
process(5)  # 12
```

## `Pipeline`

Builder for collection operations.

```python
from better_py.functions import Pipeline

result = (Pipeline()
    .map(lambda x: x * 2)
    .filter(lambda x: x > 0)
    .execute([1, -2, 3]))  # [2, 6]
```

## Example: Data Processing

```python
from better_py.functions import pipe

def parse(text): return int(text)
def validate(n): return n if n > 0 else None
def format(n): return f"Value: {n}"

result = pipe("42", parse, validate, format)
```

## Reference

| Function | Purpose |
|----------|---------|
| `pipe` | One-off transformation |
| `flow` | Reusable pipeline |
| `Pipeline` | Collection operations |
