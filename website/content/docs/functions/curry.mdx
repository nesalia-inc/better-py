## `curry(func)` - Transform to Curried Function

```python
from better_py.functions import curry

@curry
def add(x, y, z):
    return x + y + z

add(1)(2)(3)  # 6
add(1, 2)(3)  # 6
```

## `partial_right(func, *args)` - Fix Right Arguments

```python
from better_py.functions import partial_right

divide_by_2 = partial_right(lambda x, y: x / y, 2)
divide_by_2(10)  # 5
```

## `flip(func)` - Swap First Two Arguments

```python
from better_py.functions import flip

subtract_from = flip(lambda x, y: x - y)
subtract_from(10, 3)  # 7 (10 - 3)
```

## Common Pattern

```python
@curry
def fetch(api_key, endpoint, params):
    return api_call(endpoint, params, api_key)

get_users = fetch("key")("users")
get_users({"id": 1})
```
